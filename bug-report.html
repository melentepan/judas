Bug report

Official Statement from the Development Team of "Judas"
Corporate Bulletin on Current Incidents

Dear Alpha Testers,
Thank you for your feedback. The development team is actively working to resolve known issues. Below is a report on key incidents and recommended actions.

1. Installer Blocked by Windows Defender

Issue Description: Windows Defender may flag installation files as potentially harmful due to the lack of a digital signature at this stage of development. Cause: The alpha version of the product does not yet have a certified digital signature, triggering a false positive from the protection system. Solution:

  • Temporarily disable Windows Defender during installation.
  • Add the game files to Defender's exclusions list.
  • Full certification will be completed prior to the official release.

2. Crash to Desktop During Location Transitions

Issue Description: The game session terminates abruptly, returning to the desktop when moving between zones. Cause: Instability in the network code related to unfinished integration of loading systems. Status: The issue has been identified, with a fix scheduled for update v1.0.3. Workaround: We recommend saving your progress before transitioning between key locations.

3. Installer/Launcher Freezes or Fails to Start

Issue Description: The installation process is interrupted or fails to initialize. Cause: Frequent updates to the content delivery mechanism are causing version conflicts. Solution:

  • Download the latest version of the installer from the [official website].
  • Contact our support team via Discord or X for a personalized download link.

4. Game Progress Halts After "Night" Cycle

Issue Description: The game cycle does not advance to the next phase, and NPCs cease interactions. Cause: An error in processing requests to the internal AI module. Status: High-priority fix underway. The team is optimizing the algorithms. Workaround: Restart the game session—progress will be restored from the last autosave.

We apologize for any inconvenience and thank you for your participation in the testing process. For immediate assistance, please reach out to technical support via Discord or X.
Sincerely,
The "Judas" Development Team
